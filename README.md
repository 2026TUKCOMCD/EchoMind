# AI 기반 성격 분석을 활용한 소셜 매칭 시스템


### 1. 프로젝트 소개
본 프로젝트의 목적은 AI가 사용자의 실제 대화 데이터를 분석하여 성격을 자동 분석하고, 그 결과를 기반으로 성향이 잘 맞는 친구를 추천하는 시스템을 개발하는 것입니다. 설문 없이 자연스러운 언어 표현만으로 성격을 파악한다는 점에서 기존 성격검사와 차별화되며, 이를 통해 대화 속 언어 패턴이 사람의 성향을 얼마나 정확히 반영할 수 있는지 실험적으로 검증하고자 합니다. 또한 인공지능과 자연어처리 기술을 활용하여, 사용자와 잘 맞는 사람을 찾아주는 새로운 방식의 성격 분석 플랫폼을 제시합니다.

## 2. 개발 및 운용 환경

### 2.1. 개발 환경 (Development Environment)

| 구분 | 상세 내용 |
| :--- | :--- |
| **OS** | Windows 10/11 |
| **Language & Framework** | Python 3.9+, Flask |
| **Library** | OpenAI SDK, dotenv 등 |
| **Database** | Local MySQL (개발용 로컬 DB) 또는 SQLite |
| **Tools** | VS Code, Git/GitHub, Notion |

### 2.2. 운용 환경 (Operational Environment / Production)

| 구분 | 상세 내용 |
| :--- | :--- |
| **Cloud Platform** | AWS |
| **Server (Compute)** | AWS EC2 (Ubuntu Linux 24.04 LTS) |
| **Database** | Amazon RDS (MySQL) |
| **Web Server** | Nginx, Gunicorn |
| **External API** | OpenAI API (GPT-5.1 mini) |
