## 프로젝트 정보
- **팀 구성**: 3명
- **개발 기간**: 6개월
- **개발 방식**: 단계별 점진적 개발 (Phase 1 → Phase 2 → Phase 3)

## Phase 1: 성격 분석 시스템 구축 (1~2개월)

### Epic 1: 데이터 수집 및 전처리
**우선순위: 최상**

#### User Story 1.1: 카카오톡 대화 파일 업로드
- **As a** 사용자
- **I want to** 카카오톡 대화 파일(.txt)을 웹사이트에 업로드
- **So that** 내 대화 내용이 분석될 수 있도록 함
- **Story Points**: 5
- **담당**: 백엔드 개발자
- **상세 작업**:
  - [ ] 파일 업로드 API 엔드포인트 개발
  - [ ] 파일 크기 제한 설정 (예: 10MB)
  - [ ] 파일 형식 검증 (.txt만 허용)
  - [ ] 임시 저장 디렉토리 설정

#### User Story 1.2: 대화 데이터 파싱
- **As a** 시스템
- **I want to** 업로드된 카카오톡 파일에서 발화자별 대화를 추출
- **So that** 사용자의 메시지만 분석할 수 있도록 함
- **Story Points**: 8
- **담당**: 백엔드 개발자
- **상세 작업**:
  - [ ] 카카오톡 내보내기 형식 분석 (날짜, 시간, 발화자, 메시지)
  - [ ] 정규표현식으로 메시지 파싱
  - [ ] 발화자 이름 목록 추출
  - [ ] 사용자 선택 UI (여러 발화자 중 본인 선택)

#### User Story 1.3: 데이터 전처리
- **As a** 시스템
- **I want to** 불필요한 내용을 제거하고 분석 가능한 텍스트만 추출
- **So that** 정확한 성격 분석이 가능하도록 함
- **Story Points**: 5
- **담당**: 백엔드 개발자

  - [ ] 특수문자 정리

  - [ ] 성실성(Conscientiousness) 계산 공식 설계
  - [ ] 외향성(Extraversion) 계산 공식 설계
  - [ ] 친화성(Agreeableness) 계산 공식 설계
  - [ ] 신경성(Neuroticism) 계산 공식 설계

  - [ ] 가중치 조정 및 검증

## Phase 2: 매칭 시스템 구현 (3~4개월)

**우선순위: 높음**

#### User Story 5.1: 유사도 계산 함수 구현
- **As a** 시스템
- **I want to** 두 사용자 간의 성격 유사도를 계산
- **So that** 누가 누구와 잘 맞는지 판단할 수 있도록 함
- **Story Points**: 13
- **담당**: AI/데이터 개발자
- **상세 작업**:
  - [ ] 코사인 유사도 계산 함수
  - [ ] 유클리드 거리 계산 함수
