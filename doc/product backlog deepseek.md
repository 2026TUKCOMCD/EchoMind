# AI 기반 성격 분석 소셜 매칭 시스템 Product Backlog

## 📋 프로젝트 개요
**프로젝트명**: AI 기반 성격 분석 소셜 매칭 시스템  
**목적**: 실제 대화 데이터를 AI로 분석하여 성격을 자동 분석하고, 성향이 잘 맞는 친구를 추천하는 시스템  
**차별화 포인트**: 설문 없이 자연스러운 언어 표현만으로 성격 파악  
**기술 스택**: Flask, OpenAI API(GPT-4o-mini), MySQL, AWS(EC2, RDS, S3)  
**개발 기간**: 총 6개월 (2개월: 핵심 알고리즘, 4개월: UI/배포/테스트)

---

## 🎯 Epic 1: 데이터 수집 및 전처리 시스템

#### US-1.1: 사용자 인증 및 계정 관리
**As a** 사용자  
**I want** 안전하게 회원가입하고 로그인할 수 있도록  
**So that** 내 개인정보와 대화 데이터를 보호받을 수 있다

- [ ] 이메일 기반 회원가입/로그인 구현

**Tasks:**
- T1.1.1: User 모델 설계 및 MySQL 테이블 생성
- T1.1.2: Flask-Login 또는 JWT 구현
- T1.1.3: 회원가입/로그인 API 엔드포인트 개발
