# Product Backlog - AI 기반 성격 분석 소셜 매칭 시스템
**프로덕트 비전**: 카카오톡 대화 데이터 기반 AI 대화성향 추출 및 소셜 그룹 매칭 서비스

본 문서는 **AI 기반 성격 분석 소셜 매칭 시스템**의 Product Backlog입니다.
7가지 에픽(Epic)을 기준으로 분류되었으며, 각 스토리는 **[역할] - [목적] - [기능]** 구조로 작성되었습니다.


---

## 1. 에픽 (Epic) 분류

| 에픽 ID | 에픽 명 | 설명 |
|:---:|:---|:---|
| **E1** | 데이터 수집 및 전처리 | 카카오톡 대화 파싱, 정제, 개인정보 비식별화 |
| **E2** | AI 분석 엔진 (LLM) | 프롬프트 엔지니어링, JSON 구조화, 성격/스타일 분석 |
| **E3** | 매칭 알고리즘 & 큐레이션 | 유사도/보완성 기반 매칭, 가중치 산출, 그룹 추천 |
| **E4** | 백엔드 & 보안 | DB 설계, API 구축, 비밀번호 암호화 |
| **E5** | 프론트엔드 & 시각화 | 반응형 UI, 성격 리포트(레이더 차트) 시각화 |
| **E6** | 인프라 & 배포 | AWS EC2/RDS 환경 구축 및 배포 |
