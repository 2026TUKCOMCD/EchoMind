# Recommendation System Walkthrough

새롭게 구현된 **1:N 매칭 추천 시스템** 사용 가이드입니다.

## 1. 기능 소개
*   **파일**: `recommend.py`
*   **목적**: 내 프로필(`my.json`)과 후보자 폴더(`candidates/`)를 비교하여, **가장 잘 맞는 상대를 순위대로 추천**합니다.
*   **비용**: OpenAI API를 사용하지 않고, 기존 JSON 파일만으로 계산하므로 **무료**이며 **즉시 결과**가 나옵니다.

## 2. 사용 방법
터미널에서 아래 명령어를 실행하세요.

```bash
python recommend.py --my [내파일.json] --dir [후보자폴더경로]
```

### 테스트 예시 (Dummy Data)
현재 프로젝트 폴더에 테스트용 데이터가 생성되어 있습니다.

```bash
# 나: 주식 좋아하는 논리적 성향 (my_test_profile.json)
# 후보: 김주식(주식/논리), 박코딩(개발/유머), 이아이돌(감성/리액션)

python recommend.py --my my_test_profile.json --dir candidates
```

## 3. 결과 해석 (Scoring)
점수는 **100점 만점** 기준으로 산출됩니다.

1.  **대화 스타일 (50점)**
    *   어조, 화법 등이 비슷할수록 점수가 높습니다.
    *   단, '주도성'은 서로 보완적일 때(말하는 사람 + 들어주는 사람) 가산점을 받습니다.
2.  **관심사 일치 (30점)**
    *   `topics` 키워드가 많이 겹칠수록 점수가 높습니다.
3.  **AI 분석 매칭 (20점)**
    *   AI가 분석한 "잘 맞는 사람(`works_well_with`)" 조건에 부합하면 보너스 점수를 받습니다.

## 4. 참고 사항
*   **데이터 준비**: `main.py`를 통해 생성된 `.json` 파일이 필요합니다.
*   **필드**: `topics` 필드가 있으면 정확도가 높아집니다. (기존 데이터는 없어도 에러 없이 동작하지만 점수가 낮을 수 있습니다)
